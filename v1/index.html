Website Concepts

How can we build a website that people would want to share and visit multiple times?

What if the website is a little different every time you visited it?

For our first draft, I’ve created a homepage that turns the company logo into computational (coded) art. The idea is that when someone visits the homepage, they’ll encounter one of these experiences. But when they refresh or revisit the page, they’ll have a new experience that will keep the site feeling novel and fresh.

Additionally, by adapting the logo into several possible variations, we’re increasing the importance of the logo and making it more identifiable as a core brand asset.

The following variations are possible versions of the homepage artwork. We can have the site cycle through a preselected set of them. Alternatively, if there’s once concept we really like then we can go all in with that specific homepage.
